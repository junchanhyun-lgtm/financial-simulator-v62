# V62-1 Patch Notes

## 핵심 방향

V61 계좌별 분리 시뮬레이션에서 확인된 결론을 반영해 기본 엔진을 다시 통합자산 모델로 단순화했습니다.

- 국내퀀트 / 연금저축+ISA / VOO를 장기 계좌별로 강제 분리하지 않습니다.
- 연 3,800만 원 국내퀀트 → 연금저축+ISA 이전 가정을 제거했습니다.
- 연금저축은 세액공제 목적의 연 600만 원 납입 원칙으로 표시합니다.
- ISA 신규납입은 기본 0원으로 표시합니다.
- 계좌 간 이동은 총 금융자산에는 중립이므로 기본 시뮬레이션 현금흐름으로 차감하지 않습니다.

## 유지한 기능

- 결과 원인 분해 패널
- 고정 seed 검증 모드
- 시나리오별 결과 비교
- 확장 민감도 분석
- 상하위 30% 제외 평균 여유자금
- 현실 리스크 3대 지표

## 계산상 주요 변경

- 은퇴 전 포트폴리오 전환은 연 3,800만 원 이체 기반으로 자동 낮추지 않습니다.
- 은퇴 전에는 선택한 은퇴 전 기대수익률·변동성, 은퇴 후에는 은퇴 후 기대수익률·변동성을 적용합니다.
- 국내퀀트 규모 페널티는 총자산 중 초기 국내퀀트 비중을 기준으로 추정 적용합니다.
- t분포 난수가 만드는 -100% 이하 비현실적 연간수익률을 막기 위해 연간수익률 하한 -95%를 적용합니다.
